Frequently asked Questions


  • What is this website's purpose?

    • The Website is a informal movie page built from gathered movie databases like Omdb. With the website you can search for any movie and even some games and get all the information you need.


  • How do you search for movies?

    • Simply click on the searchbar and search for the movie you want to find, then click on the searchbar and you will get results simular to what you wrote. For an example, if you type star, you will get a list of movies that begins with star such as Star Wars or Star Trek.


  • What Api's is used on the website?

    • This website is built for educational purposes and the Api's used are the Omdb Api by Brian Fritz and the Youtube Api from Google
